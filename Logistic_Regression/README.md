# Customer Churn Predictor

ML-powered system to predict customer churn risk for subscription/telecom businesses.

## 🔗 Live Demo
- **Web App:** https://customer-churn-predictor32.streamlit.app/
- **API Docs:** https://logistic-regression-pwx8.onrender.com/docs

## 📊 Model Performance
- Algorithm: Logistic Regression (with class balancing)
- ROC-AUC: 0.834
- Churn Recall: 79% (catches 4 out of 5 actual churners)

## 🛠️ Tech Stack
- scikit-learn, pandas, FastAPI, Streamlit, Plotly

## 🎯 Business Problem
Customer churn costs businesses significant recurring revenue. This tool identifies
at-risk customers before they leave, enabling proactive retention strategies.

## 📈 Key Findings
- Month-to-month contracts have highest churn risk
- Short tenure customers are most likely to churn
- Higher monthly charges correlate with increased churn
